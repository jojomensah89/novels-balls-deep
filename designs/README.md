## Design Guidelines

- **Style**: Modern, clean, dark-mode first
- **Colors**: Deep purples/blues with accent colors
- **Typography**: Clean sans-serif, highly readable
- **Mobile**: Responsive, mobile-first approach
- **Inspiration**: novelbin.com, novelhi.me (but more modern)
